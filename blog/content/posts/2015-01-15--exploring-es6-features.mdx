ECMAScript¬†2015 (ES6) adds modern syntax, safer defaults, and built‚Äëin tools for async and modular code.

Transpile with **Babel**:

```bash
npm init -y
npm install --save-dev @babel/core @babel/cli @babel/preset-env
echo '{ "presets": ["@babel/preset-env"] }' > .babelrc
npx babel src --out-dir dist
```

Place ES6 files in **src/**, run the command, and load the output from **dist/** in older browsers.

### let and const

```js
for (let i = 0; i < 3; i++) {
  // i scoped to loop only
}
const API_URL = '/api';
```

---

### Arrow functions

```js
const doubled = [1, 2, 3].map(n => n * 2);
button.addEventListener('click', () => this.save());
```

---

### Template literals

```js
const user = 'Luiz';
const msg = `Hi, ${user}
Welcome to ES6.`;
```

---

### Destructuring

```js
const [red, green] = ['#f00', '#0f0'];

const user = { id: 7, role: 'admin' };
const { id, role } = user;
```

---

### Default, rest, spread

```js
function greet(name = 'guest') { return `Hi ${name}`; }

const nums = [1, 2, 3];
sum(...nums); // spread into args

function logAll(...args) { console.log(args); }
```

---

### Enhanced object literals

```js
const name = 'x';
const obj = {
  name,                // shorthand
  [`prop_${name}`]: 1, // computed key
  toString() { return this.name; }
};
```

---

### Classes

```js
class Counter {
  constructor(start = 0) { this.count = start; }
  inc() { return ++this.count; }
}
```

---

### Modules

```js
// math.js
export const PI = 3.14;

// app.js
import { PI } from './math.js';
console.log(PI);
```

*Run through Babel or a bundler to use modules in the browser.*

### Iterators and for..of

### Generators

```js
function* idMaker() {
  let id = 0;
  while (true) yield id++;
}
const gen = idMaker();
gen.next().value; // 0
```

### Proxies

```js
const monitor = new Proxy({}, {
  get(target, prop) {
    console.log('read', prop);
    return target[prop];
  }
});
monitor.x = 5;
monitor.x; // logs "read x"
```

### Reflect¬†API

```js
const obj = {};
Reflect.set(obj, 'x', 1);
Reflect.get(obj, 'x'); // 1
```

### Tail‚Äëcall optimisation

```js
function factorial(n, acc = 1) {
  return n === 0 ? acc : factorial(n - 1, n * acc);
}
```

Tail‚Äërecursive functions can run without growing the call stack in compliant engines.
