# Building APIs with Next.js

Next.js provides a powerful way to build APIs using API routes. Let's explore how to create and use them effectively.

## Creating an API Route

```typescript
// pages/api/hello.ts
import type { NextApiRequest, NextApiResponse } from 'next';

type Data = {
  message: string;
  timestamp: string;
};

export default function handler(
  req: NextApiRequest,
  res: NextApiResponse<Data>
) {
  res.status(200).json({
    message: 'Hello from the API!',
    timestamp: new Date().toISOString()
  });
}
```

- RESTful endpoints
