# Getting Started with Next.js

Next.js is a powerful React framework that makes building full-stack web applications simple and efficient. In this guide, we'll explore the key features that make Next.js special.

## Why Next.js?

```javascript
// Example of a Next.js page component
export default function HomePage() {
  return (
    <div>
      <h1>Welcome to Next.js!</h1>
      <p>This is a simple page component.</p>
    </div>
  );
}
```

## Key Features

- Server-side rendering
- Static site generation
- API routes
- File-based routing
- Built-in CSS support
